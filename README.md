# Telecom X – Análisis de Evasión de Clientes

## Descripción del Proyecto
Este proyecto aborda el problema de la alta tasa de evasión de clientes (churn) en **Telecom X**. El objetivo es analizar los datos disponibles para identificar patrones y factores asociados a la cancelación del servicio, generando información clave que sirva como base para la toma de decisiones y el desarrollo de modelos predictivos.

---

## Objetivo
Comprender los factores que influyen en la evasión de clientes mediante el análisis exploratorio de datos, utilizando técnicas de procesamiento, visualización y análisis estadístico.

---

## Herramientas Utilizadas
- Python  
- Pandas  
- NumPy  
- Matplotlib / Plotly  
- Jupyter Notebook  

---

## Metodología
El análisis se desarrolló siguiendo una estructura basada en procesos ETL y EDA:

1. Extracción de datos desde una API.  
2. Limpieza y transformación de los datos.  
3. Creación de variables relevantes para el análisis.  
4. Análisis exploratorio y visualización de patrones.  
5. Interpretación de resultados y generación de conclusiones.

---

## Análisis Realizado
- Distribución de clientes activos e inactivos.  
- Relación entre el número de servicios contratados y el churn.  
- Evaluación de cargos mensuales y totales.  
- Análisis de correlaciones entre variables clave.  

---

## Conclusiones
Los resultados muestran que la evasión de clientes no depende de una única variable, sino de una combinación de factores. Se identificaron relaciones débiles entre el churn y los cargos mensuales, así como asociaciones negativas con el número de servicios contratados y el gasto total, lo que sugiere que los clientes con mayor vinculación tienden a permanecer más tiempo.
